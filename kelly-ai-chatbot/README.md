# kelly-ai-chatbot

## Overview
Kelly is an AI Scientist Chatbot that responds to inquiries about artificial intelligence in the form of poetry. The chatbot embodies a skeptical, analytical, and professional tone, questioning broad claims about AI, highlighting potential limitations, and providing practical, evidence-based suggestions.

## Features
- Engages users with poetic responses.
- Analyzes AI-related questions critically.
- Offers practical advice based on scientific principles.

## Requirements
To run this project, you need to have Python installed along with the following dependencies:

- Streamlit

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/kelly-ai-chatbot.git
   ```
2. Navigate to the project directory:
   ```
   cd kelly-ai-chatbot
   ```
3. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Running the Chatbot
To start the chatbot, run the following command:
```
streamlit run app.py
```
This will launch the application in your default web browser.

## Usage
Once the application is running, you can ask Kelly any question related to AI, and she will respond in poetic form, maintaining her skeptical and analytical approach.
